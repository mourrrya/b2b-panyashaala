# Business Information - Color Palette

## Overview

Generic color scheme for business-related information sections and components.

## Primary Colors

| Color  | Tailwind     | Hex       | Purpose          |
| ------ | ------------ | --------- | ---------------- |
| Blue   | `blue-500`   | `#3B82F6` | Primary accent   |
| Indigo | `indigo-500` | `#6366F1` | Secondary accent |

## Secondary Colors

| Color        | Tailwind    | Hex       | Purpose           |
| ------------ | ----------- | --------- | ----------------- |
| Blue Light   | `blue-50`   | `#EFF6FF` | Light backgrounds |
| Indigo Light | `indigo-50` | `#EEF2FF` | Light backgrounds |
| Blue Dark    | `blue-600`  | `#2563EB` | Darker text/icons |

## Neutral Colors

| Color     | Tailwind    | Hex       | Purpose        |
| --------- | ----------- | --------- | -------------- |
| Slate 800 | `slate-800` | `#1E293B` | Primary text   |
| Slate 600 | `slate-600` | `#475569` | Secondary text |
| Slate 700 | `slate-700` | `#334155` | Tertiary text  |

## Background Colors

| Color     | Tailwind    | Hex       | Purpose                     |
| --------- | ----------- | --------- | --------------------------- |
| Slate 50  | `slate-50`  | `#F8FAFC` | Light backgrounds           |
| Slate 100 | `slate-100` | `#F1F5F9` | Slightly darker backgrounds |
| White     | `white`     | `#FFFFFF` | Primary background          |

## Border Colors

| Color     | Tailwind    | Hex       | Purpose        |
| --------- | ----------- | --------- | -------------- |
| Slate 100 | `slate-100` | `#F1F5F9` | Light borders  |
| Slate 200 | `slate-200` | `#E2E8F0` | Medium borders |

## Gradient Combinations

- **Header**: `from-blue-50 to-indigo-50`
- **Accent**: `from-blue-500 to-indigo-500`
- **Subtle**: `from-slate-50 to-slate-100`

## Usage Guidelines

- Use blue/indigo for primary sections and accent elements
- Use slate tones for text and neutral backgrounds
- Apply gradients for visual hierarchy and depth
- Maintain white backgrounds for content areas
- Use borders with slate-200 for definition
