# Account Overview - Color Palette

## Overview

Generic color scheme for account overview and statistics display sections.

## Primary Colors

| Color  | Tailwind     | Hex       | Purpose          |
| ------ | ------------ | --------- | ---------------- |
| Purple | `purple-500` | `#A855F7` | Primary accent   |
| Pink   | `pink-500`   | `#EC4899` | Secondary accent |

## Gradient Combinations

- **Header**: `from-purple-50 to-pink-50`
- **Accent**: `from-purple-500 to-pink-500`
- **Type 1**: `from-emerald-50 to-teal-50`
- **Type 2**: `from-blue-50 to-indigo-50`
- **Type 3**: `from-amber-50 to-orange-50`
- **Type 4**: `from-purple-50 to-pink-50`

## Usage Guidelines

- Use purple/pink for section headers and main accent
- Apply four distinct colors for stat cards to create visual variety
- Use emerald for Type 1 items (primary metrics)
- Use blue for Type 2 items (secondary metrics)
- Use amber for Type 3 items (tertiary metrics)
- Use purple for Type 4 items (quarterly/annual data)
- Combine light and dark shades for depth
- Apply borders with lighter shades for definition
